팝업 가이드

1. 윈도우 팝업

HTML 예시

	< a href="guide.pop.asp" onclick="window.open( this.href, '_blank', 'width=850, height=740, top=0, left=0, scrollbars=yes' ); return false">버튼< /a>

TEST

팝업띄우기 버튼
  • * 사용법 : href (팝업주소) 수정, onclick="..."복사하여 옵션값 변경
  • * 주의 : 팝업창의 실제 width-height 값 잘보고 적용 ( border나 padding 포함 )
  •    - width-height 이 1px이라도 맞지 않으면 스크롤이 발생할 수 있음
  •    - 개발자도구(F12) 로 border나 padding를 포함한 실제 width-height값 확인하기
  • * < area > 태그 에서도 동일하게 사용가능
  • 옵션 설명
  • this.href : 현재 a의 href (팝업창 주소)
  • _blank : 새 창으로 로드
  • width, height : 팝업창의 가로, 세로 값
  • top, left : 팝업창 띄워질 위치(윈도우 왼쪽 상단 기준)
  • scrollbars: 스크롤바 유무 (yes=생김, no=안생김)
  • "scrollbars=no"로 하더라도 1px 안맞으면 스크롤 생길 수 있음
  • 참고사이트 : w3
  • 참고사이트 : 옵션설명 및 Javascript 코드 예제

2. 윈도우 팝업 "IE 여백 문제"해결 ( width/height 값을 .popup에 지정 후 아래 스크립트 사용)


<!-- 스크롤 없는 팝업 예시 -->
<body>	

	<style>
		html,body { overflow: hidden !important; }
	</style>
	<div class="popup" style="overflow: hidden; width: 600px; height: 500px;"> <!-- widt/height 값 지정 필수  -->
		<!--  팝업내용  -->
	</div>


	<script type="text/javascript"src="http://img.megamd.co.kr/common/js/jquery.min.js"></script>
	<script>
		//팝업 리사이징 ( IE 팝업 사이즈 문제 해결 ) 
		$(window).load(function() {
			var popWidth = $('.popup').outerWidth() + (window.outerWidth - window.innerWidth);
			var popHeight = $('.popup').outerHeight() + (window.outerHeight - window.innerHeight);
			window.resizeTo( popWidth, popHeight ); //resize 
		});
	</script>

</body>	
<!-- 스크롤 없는 팝업 예시 -->
