2. 옵션값 설명

Name Description Option / Value Default
data-mode 슬라이드 동작방법 [horizontal,vertical,fade] horizontal
data-maxSlides 최대 보여지는 갯수 [1] 1
data-minSlides 최소 보여지는 갯수 [1] 1
data-slideMargin 슬라이드 사이 margin 값 [0] 0
data-delay 슬라이드 딜레이 설정 [4000] 4000
data-speed 슬라이드 속도 설정 [400] 400
data-auto 자동슬라이드 여부 [true,false] false
data-pager 페이지 동그란 버튼 설정 여부 [true,false] false
data-pagercustom 페이징 직접생성&연결 [class명] false
data-controls 자동 슬라이드 컨트롤 버튼 설정 여부(화살표) [true,false] false
data-moves 슬라이드 몇개씩 움직일지 [1] 1
data-loop 무한 루프 [true,false] true
data-ticker 흐르는 슬라이드 [true,false] false
data-tickerHover 흐르는 슬라이드 hover시 멈춤 [true,false] false
data-randomStart 랜덤값 [true,false] false

<script>
	$(document).ready(function () {
		$('.js_tab_type li a').on('click', function (e) {
			var targetId = $(this).attr("href");
			if ($(this).attr("target") !== "_blank") {
				e.preventDefault();
				$(this).parent('li').addClass('on').siblings().removeClass('on');
				$(targetId).addClass('on').siblings().removeClass('on');
				__globalBxslider.reloadSlider();
			} else {
				return true;
			}
		});
	});
</script>
