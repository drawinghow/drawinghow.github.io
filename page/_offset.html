절대좌표 상대좌표 확인하기

1. 절대좌표 offset() : 윈도우 최상단부터 섹션까지의 높이값을 구하고 싶을때

JS

	<script type="text/javascript"> 

		$(window).load(function(){
			var Height = $(' #아이디 또는 .클래스명 ').offset().top;        // #아이디 또는 .클래스명 변경
			alert( Height );            
		});

	</script>

2. 상대좌표 position() : 부모 요소의 위치를 기준으로 값을 얻는 상대좌표(상대위치)

JS

	<script type="text/javascript"> 

		$(window).load(function(){
			var Height = $(' #아이디 또는 .클래스명 ').position().top;        // #아이디 또는 .클래스명 변경
			alert( Height );            
		});

	</script>
